### Overview

* [Jade](http://jade-lang.com/) for HTML templating
* [Sass](http://sass-lang.com/) for CSS pre-processing
* [Grunt](http://gruntjs.com/) for task running


### Setup

**Requirements:**

* [node.js](https://nodejs.org/)
* [npm](https://www.npmjs.com/)

**Initalise:**

1. `npm install`
2. `grunt`
